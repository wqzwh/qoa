## 介绍

简单实现koa基本功能，仅供学习参考，了解koa的核心思想以及实现过程，不能线上使用哦

## 环境要求

依赖 node v7.6.0 或 ES2015及更高版本和 async 方法支持。

## 实现功能

主要依据`koa`官网介绍，实现简易核心功能，基本实现以下几个功能

- [ ] use
- [ ] listen
- [ ] callback
- [ ] keys
- [ ] context 上下文
- [ ] 错误处理
- [ ] response
- [ ] Request
- [ ] Middleware
- [ ] next
